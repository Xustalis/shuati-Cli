# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-02-26

### Added
- **Bootstrap**: Initial project setup with unified version v0.0.1.
- **Build**: Fixed CMake configuration and build issues.
- **CI**: Fixed GitHub Actions CI/CD workflows.
- **Core Features**:
  - Multi-Platform Crawler: LeetCode (GraphQL), Codeforces, Luogu, Lanqiao
  - Local Judge Engine: sandbox execution for C++ and Python
  - AI Coach: LLM integration for error diagnosis
  - Memory System: SM2 algorithm for spaced repetition
  - Interactive REPL mode with auto-completion
